Protocols API

The protocols module provides interfaces for interacting with various blockchain protocols and DeFi platforms. Each protocol implementation handles the specific requirements and APIs for that platform.

Main Protocols Module

Hyperliquid Protocol

airdrops.protocols.hyperliquid.stake_rotate(exchange_agent, info_agent, current_validator_address, new_validator_address, amount_wei)[source]

Rotates staked HYPE from one validator to another on Hyperliquid.

This involves undelegating from the current validator and then delegating the same amount to a new validator using the Hyperliquid API.

Return type:

bool

Parameters

exchange_agenthyperliquid.exchange.Exchange

An initialized Exchange agent from the hyperliquid-python-sdk.

info_agenthyperliquid.info.Info

An initialized Info agent from the hyperliquid-python-sdk, used for optional pre-checks like validator info or current delegations.

current_validator_addressstr

The 0x-address of the validator to undelegate from.

new_validator_addressstr

The 0x-address of the validator to delegate to.

amount_weiint

The amount of HYPE in wei to rotate.

Returns

bool

True if both undelegation and delegation were successful, False otherwise.

Examples

>>> # Conceptual example, requires live initialized agents
>>> # from hyperliquid.exchange import Exchange
>>> # from hyperliquid.info import Info
>>> # from hyperliquid.utils import constants
>>> # config = {"wallet": "0x...", "private_key": "...",
>>> #           "base_url": constants.MAINNET_API_URL}
>>> # exchange_agent = Exchange(config["wallet"], config["private_key"],
>>> #                           base_url=config["base_url"])
>>> # info_agent = Info(base_url=config["base_url"])
>>> # stake_rotate(exchange_agent, info_agent, "0xValidatorA",
>>> #              "0xValidatorB", 1000000000000000000)  # 1 HYPE
True  # If successful
airdrops.protocols.hyperliquid.vault_cycle(exchange_agent, info_agent, user_address, vault_address='0xdfc24b077bc1425ad1dea75bcb6f8158e10df303', min_deposit_usd_units=20000000, max_deposit_usd_units=40000000, min_hold_seconds=1800, max_hold_seconds=5400)[source]

Performs a vault cycle: deposit, hold, and withdraw from a Hyperliquid vault.

This function deposits a random amount of USDC into a specified vault, holds it for a random duration, and then withdraws the full equity. Operations use the hyperliquid-python-sdk.

Return type:

bool

Parameters

exchange_agenthyperliquid.exchange.Exchange

An initialized Exchange agent from the hyperliquid-python-sdk.

info_agenthyperliquid.info.Info

An initialized Info agent from the hyperliquid-python-sdk.

user_addressstr
